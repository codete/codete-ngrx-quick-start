# Installation & build

1. Install firedev:
```
npm i -g firedev
```

2. Start build:
```
cd codete-ngrx-quick-start
firedev build:watch main
```

3. Start server:
```
npx ts-node run.json
```
or in Visual Studi code select proper debug task and press F5 on keyboard

![](proper-debug.png)

![](vscode-debug.png)

4. Open browser: http://localhost:4200
